API megoldások

Integrálja a munkafolyamataiba a globális piacokat az IBKR API megoldásaival.

Emelje új szintre a kereskedési élményét: férjen hozzá globális piacokhoz az IBKR API-kkal. API felületeink fejlett kereskedési, számlakezelési és jelentéstételi eszközöket kínálnak magánszemélyeknek, intézményeknek, Fintech vállalatoknak és vállalati ügyfeleknek. Kezdje el az integrálást még ma!

Az IBKR által kínált előnyök

  • Kereskedjen részvényekkel, opciókkal, határidős ügyletekkel, devizákkal, kötvényekkel és alapokkal 160 piacon a világ 36 országában.
  • Férjen hozzá az IBKR számla összes funkciójához, beleértve a fedezeti kereskedést / short eladást és a készpénzegyenleg után fizetett magas kamatokat.
  • Tekintse meg a valós idejű számla- és teljesítményadatokat, többek között a pozíció és fedezeti információkat
  • Streameljen valós idejű vagy múltbéli piaci adatokat

Fedezze fel az IBKR API-kban rejlő lehetőségeket

  • Fejlesszen új mobil-, webes vagy asztali alkalmazást
  • Bővítse új kereskedési funkciókkal a meglévő technológiai infrastruktúráját
  • Automatizálja a haladó szintű programozott befektetést és kereskedést
  • Nyújtson be programozott megbízásokat, beleértve a haladó megbízástípusokat és algoritmusokat, és kereskedjen kiterjesztett/napon túli kereskedési időben
  • Tervezzen teljesen egyedi számlanyitási és kereskedési folyamatot
  • Használja a digitális számlakezelést, beleértve a számlafeltöltést és a pozíciótranszfereket is (csak intézmények)
  • Hozzon létre személyre szabott szoftvert az IBKR-számlája használatához és a kereskedéshez

API kínálatunk

Ügyfeleink maguk választhatják ki a használni kívánt API megoldásokat, igény szerint használhatnak csak egy, vagy akár sok különböző API funkciót.

Web API

Vegye igénybe cégünk modern REST API megoldását, amely a lehető legtöbb funkciót felöleli, többek között a számlanyitási/-kezelési, finanszírozási, banki, jelentéstételi és kereskedési funkciókat. Férjen hozzá a magánszemélyeknek kínált számlákhoz és a pénzügyi tanácsadóknak kínált alszámlastruktúrához. Használja a WebSocket streaminget a valós idejű piaci adatokhoz és fontos értesítésekhez.

FIX

Az IBKR iparági sztenderd FIX kapcsolata lehetővé teszi az intézmények számára olyan kereskedési rendszerek kialakítását, amelyek kihasználják az általunk kínált nagy sebességű megbízásirányítást és piacmélységet. A FIX közvetlen és skálázható hozzáférést kínál az IBKR kereskedési rendszerhez VPN-nel, extranettel, dedikált vonallal vagy keresztkapcsolattal.

FIX API dokumentáció

TWS API

Ez a kereskedésorientált API lehetővé teszi az alkalmazásfejlesztést C++, C#, Java, Python, ActiveX, RTD vagy DDE programozási nyelven, előre összeállított könyvtárakkal, amelyekkel automatizálhatja a TWS UI funkciókat vagy akár saját felületet is fejleszthet. A TWS API ideális lehet az Ön számára, ha az ügyfél gateway-en keresztül szeretne hozzáférni *nagyobb kereskedési volumenhez kisebb sávszélesség használatával*.
