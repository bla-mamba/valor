Options Trading

Enhanced Insight.
Efficient Interpretation.

  • Options Lattice
  • Options Chains
  • Options Analysis
  • Options Wizard

Traders can view a chart depicting the 30-day price history for a stock alongside an array of bubbles measuring relative volume or value for key metrics across expirations and strike prices and compare bubble sizes for calls or puts indicating option volume, open interest or implied volatility. Such metrics allow the user to understand where options activity is taking place both in today’s market and building over time by focusing on volume and open interest.

Learn About Option Lattice

Option Chains allow traders to view and trade single or multi-leg options on stocks, futures, and index instruments. Configuring strategies like Calendar Spread, Vertical Spread or Diagonal Spread and adding additional data such as VWAP, Option Volume and Greeks is straightforward.

Learn About Option Chains

The Option Analytics Window lets option traders visualize changes in an option's price with respect to unit changes in Greek values. Traders use the Options Analytics window to:

  • Create interactive curves to see immediately how changing volatility assumptions, stepping closer to expiration and adjusting the price of the underlying impacts the value of an option line.
  • Specify close values by increasing or decreasing the implied volatility and reducing the time to expiry.
  • Create dynamic curves based on different date scenarios.
  • Modify the strike prices, expiries and exchanges using easily accessible controls.
  • View Greeks and Put+Call P&L curves together or separately.

The Options Wizard helps traders, including those who may be newer to options, identify and evaluate potential options strategies quickly and with greater clarity. Use Options Wizard to:

  • Create Options Strategies: Answer a few questions to define your forecast and the Options Wizard provides common strategies aligned with your forecast.
  • Visualize Outcomes: Options Wizard provides side-by-side comparisons for multiple strategies, allowing you to compare profit/loss, break-even points and risk/reward profiles.
  • Customize Assumptions: Define key assumptions like target stock price, time horizon and volatility outlook, and the Options Wizard recalculates outcomes accordingly.
  • Quickly Execute Strategies: Finalize your strategy and send orders directly to the market from the same interface.
